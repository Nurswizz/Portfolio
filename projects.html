The Way of Warrior

2D Adventure Game about Kazakh History. A plot starts from founding of Kazakh Khanate.
A main character is 14-years-old guy who wants to become one of the strongest men in
the whole history. The game contains not only Kazakh culture, but other characters such as
elfs, goblins, and etc.

Engine: Unity
Programming Language: C#
Skills used: Mathematics, Algorithms, Data Structures, Basic Physics
